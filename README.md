# `xtools_js`

一款实用的 JS 工具库

* [文档地址](https://borenxue.github.io/xtools_js/)
* [npmjs.com 链接](https://www.npmjs.com/package/xtools_js)

<br/>

> 安装依赖


```shell
npm install -S xtools_js
```

```javascript

// ...
```
